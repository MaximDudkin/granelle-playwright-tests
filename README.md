# UI Автотесты на Playwright для Granelle

Этот репозиторий содержит набор UI-автотестов для сайта группы компаний **«Гранель»**. Проект демонстрирует навыки автоматизации тестирования e-commerce функционала недвижимости, работы с динамическими списками и избранным.

## 🔗 Стек технологий

- **Framework:** Playwright
- **Language:** TypeScript

## 📋 Какие проверки реализованы:

1.  **Навигация:** проверка работы ссылок в шапке (header) на главной странице.
2.  **Проекты на главной:** проверка отображения ровно 22 карточек жилых комплексов.
3.  **Избранное:** добавление квартиры в список «Избранное» и её последующее удаление на странице подбора.
4.  **Списки квартир:** проверка соответствия количества карточек квартир (22 шт.) на странице поиска.
5.  **Карточка квартиры:** переход в детальную карточку объекта и проверка доступности кнопки бронирования.

## 🛠 Установка и запуск

**1. Клонируйте репозиторий:**

```bash
git clone https://github.com/MaximDudkin/granelle-playwright-tests.git
```

**2. Установите зависимости:**

```bash
npm install
```

**3. Запустите тесты:**

```bash
npx playwright test
```

**4. Запуск в UI-режиме (с визуализацией):**

```bash
npx playwright test --ui
```
